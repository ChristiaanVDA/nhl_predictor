NHL Playoff Predictor: Data

We obtained NHL team data by scraping the website 'Puck On Net' using Python, then used Pandas to organize the data before writing the output as csv files. We downloaded 82-game data for each NHL team for the seasons ending in 2016, 2017, and 2018. Data included the “advanced stat” measures of Fenwick and Corsi, as well as the component data of goals, shots on goal, missed shots, and blocked shots.

We also took information from the NHL's official website.

For the model, we limited the data to cumulative statistics at the 20-, 40-, 60-, and 82-game marks for each season. For the prediction, we only used cumulative game results through 60 games.
